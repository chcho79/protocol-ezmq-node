Class: EZMQSubscriber

EZMQSubscriber(ip, port, subCB, subTopicCB)

This class represent EZMQSubscriber. It contains APIs to start/stop subscriber and to subscribe for data.

ProtoType of subCB :- function subCallback(event) {}
ProtoType of subTopicCB :- function subTopicCallback(topic, event){}

In callback event will be EZMQEvent/EZMQByteData
